# IPPASS 学習マップ — ITパスポート対策サイト(シラバスVer.6.5対応)

学習科学(先行オーガナイザー・アクティブリコール・インターリーブ・メタ認知)をUIに組み込んだ、静的な学習サイト。サーバー不要で、`index.html` をブラウザで開くだけで動きます(GitHub Pages等での公開も可)。

## 構成

```
itpassport-site/
├── index.html        トップ: 路線図学習マップ(29駅・進捗点灯)
├── quiz.html         分野別演習(即時フィードバック+解説+講義リンク)
├── exam.html         模擬テスト(タイマー・1000点換算・総合600/分野別300判定・見直し)
├── glossary.html     用語辞典(検索・分野フィルタ・新出タグ)
├── dashboard.html    学習の記録(講義進捗・分野別正答率・模擬テスト履歴)
├── assets/
│   ├── style.css     デザインシステム(分野色: 山吹/青磁/瑠璃、完了=朱印)
│   └── app.js        localStorage進捗管理・出題サンプリング
├── data/             ★コンテンツはすべてここに分離(シラバス改訂時はここだけ更新)
│   ├── curriculum.js       29回のカリキュラム
│   ├── questions.js        基本問題プール
│   ├── questions_r0X_*.js  年度別過去問(令和3〜8年度、全600問。分野ごとにS/M/Tへ分割)
│   └── terms_*.js          用語辞典(頭文字a〜kで分割)
├── lessons/          講義ページ(第01〜29回、全回作成済み)
├── KAKOMON_INDEX.md  過去問の出題トピック索引・収録方法メモ
└── COURSE.md         コース設計書(ミッション・カリキュラム・進捗)
```

`test_data/`(過去問PDF原本)はコンテンツ生成用の作業ファイルのため`.gitignore`で除外している。サイトの動作には不要。

## データの追加方法

### 問題を増やす(data/questions_r0X_s/m/t.js など)
```js
{ id:"T17", cat:"T", lesson:24, q:"問題文",
  opts:["ア","イ","ウ","エ"], ans:2, exp:"解説" }
```
- `cat`: S=ストラテジ / M=マネジメント / T=テクノロジ
- `ans`: 正解の添字(0〜3)
- `lesson`: 復習リンク先の講義回
- IPAの過去問は出典明記のうえ利用可。追加時は `exp` に「出典: 令和◯年度 問◯」を含めることを推奨
- 模擬テストは分野比率(35/20/45)で自動サンプリングするので、問題を増やすほど本番に近づく
- 新しい年度ファイルを追加した場合は、`quiz.html`・`exam.html`の`<script>`読み込みと`assets/app.js`の統合配列にファイル名を追加すること

### 用語を増やす(data/terms_*.js)
```js
{ term:"用語", en:"English", cat:"T", star:true, lesson:17, def:"説明" }
```

## 講義ページ

`lessons/lessonNN.html`(第01〜29回)は全回作成済み。差し替える場合は末尾の「完了ボタン」ブロック(`IPSS.initLessonPage(N)`)を残すと進捗連動が維持される。

## 学習データの保存について

進捗・正答率・模擬テスト履歴はすべて閲覧者のブラウザのlocalStorageに保存され、外部には送信されない。端末を変えるとリセットされる点は配布時に明記のこと。
